# ⚡ TypeSlayer — Typing Battle Arena

A fast-paced typing battle game playable directly in your browser.
No installs. No accounts. Just type.

## 🎮 Game Modes

- **⚔️ Solo Quest** — Race against an AI bot in Easy (35 WPM), Medium (60 WPM), or Hard (90 WPM) difficulty.
- **🏹 1v1 Race** — Get matched with an opponent and race to finish a passage first.
- **💀 Boss Battle** — A powerful boss attacks at regular intervals. Type words correctly to deal damage and build combos before your HP hits zero.

## 📦 Run Locally

1. Download or clone this repo
2. Open `index.html` in any browser
3. Done ✅

```bash
git clone https://github.com/ASULTayan/typeslayer.git
cd typeslayer
# open index.html in browser
```

## 🚀 Deploy on GitHub Pages

1. Push all files to a GitHub repo
2. Go to **Settings → Pages**
3. Set source to **Branch: main**
4. Your game is live at `https://ASULTayan.github.io/typeslayer`
